To run node backend:
1. Download and install XAMPP Control Panel
2. Start mySQL and APACHE services
3. Open a terminal
4. cd server
5. npm install
6. npm start

To run python backend:
1. cd video-computation
2. run pip install -r requirements.txt
3. python computation.py

To run frontend:
1. cd concussion-app
2. npm install
3. npm run dev
